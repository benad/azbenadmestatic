With the first update to Mac OS X 10.8 "Mountain Lion" released this week (10.8.1), am I ready to update my work laptop? After using it on my old laptop for a little while, I think so, but only after a thorough backup, just in case.

The same can be said for VMware Fusion 5. It seems like an incremental upgrade, but doing so from version 4 would give me the "Pro" version with an advanced virtual network editor and a license VMware Player.

Combining both upgrades could be risky, so if anything doesn't work I'll revert to a backup. Otherwise, it will be a small but nice incremental upgrade for this fall.
