This afternoon I was preparing the CSS of my new blog location when I notice how better web design tools are now compared to ten years ago. Firebug, the extension for Firefox has always been a leader, though I often used external software for CSS editing. With Safari came a powerful set of built-in tools that can inspect any page element and show how and why it was styled a certain way. Google Chrome copied that also (since it inherited from WebKit, the engine of Safari). Today I combined both Espresso 2 from MacRabbit, powerful HTML and CSS with live preview and Firefox + Firebug, both that I haven’t used in a while. Wow. This was so much easier than plain-text editors with the CSS spec open… Maybe it’s time to revamp my web site’s (anemic) style…
