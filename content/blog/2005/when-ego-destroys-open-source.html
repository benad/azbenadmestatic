A quick post...

I always had some doubts concerning JBoss' (the corporation) agressivity against other open-source projects , especially the spring framework. But doing a partnership with Microsoft to compete against IBM for the server space? Make Hibernate work better with MSSQL? Better integrate Java and .Net?

Oh, and suing the heck out of everyone using the word "JBoss", especially since JBoss Inc. want to keep a monopoly of offering JBoss related services?

Hmmm... So sad to see good open-source projects cought up in a few people's ambition and ego...

Note to self: don't be like that, no matter what happens.
