A quick post about a great interactive version of the du command. It's called "NCurses Disk Usage" (ncdu), and works directly on the console by using ncurses. In the past, I used to collected disk usage data from GUI-less servers using du then transferring the file back to a GUI-enabled machine and display it using either xdiskusage or KDirStat. Cumbersome to say the least.

Nice touches of ncdu includes its great portability, and the fact that its web site has the most comprehensive list of graphical du replacements I've seen.
