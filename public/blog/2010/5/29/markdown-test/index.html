This is actually a test post. Some context: For the long term, I am
planning to move my blog to Posterous since I always end up
writing my posts in Evernote and sending them by email.
Livejournal has an email interface, but quite frankly
writing posts with HTML tags on an iPhone is challenging.

Recently, Posterous started supporting the Markdown syntax, which
is exactly the kind of syntax I wanted for email posting. So, I’ve set
up a “mirror” Blog at benad-blog.posterous.com that autoposts to benad.livejournal.com. I’ll keep my default Posterous site for micro-blogging.
