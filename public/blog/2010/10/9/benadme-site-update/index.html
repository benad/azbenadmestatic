I haven’t updated my web site in a while, but that doesn’t mean I was totally idle.

I tried to re-style the page with a custom font loaded with CSS3 font-face but it miserably failed on my iPad when using proper font variants (bold, italics) that would have gracefully degraded in older browsers. In a way, without an iPad, I would have never known that my site could consistently crash its browser…

I also streamlined my work process by creating a new Mercurial repository on Bitbucket so that I can sync and backup my source repository easily, and made a quick-and-dirty Markdown to my-XML-format converter.

I’m seriously starting to dislike Evernote’s editor on the iPad and iPhone, so I’ll be moving to GoodReader’s built-in text editor, write the page in Markdown (at least the subset supported by my converter) and use my iDisk as an in-between place to transfer the file to my laptop (I could use Dropbox too).
